О нас

Данный проект представляет собой онлайн магазин, созданный для дисциплины "Основы проектной деятельности".

Основные особенности проекта:

  • Макет сайта: Наш сайт разработан с использованием технологий HTML, CSS и JavaScript для создания удобного и привлекательного интерфейса.

  • База данных PostgreSQL: Мы используем PostgreSQL для хранения информации о товарах, пользователях, заказах и других важных данных, обеспечивая надежное и эффективное хранение информации.

  • Интеграция с FakeStore API: Для расширения ассортимента товаров и обеспечения актуальной информации о продуктах мы интегрировали FakeStore API, который предоставляет доступ к реалистичным товарам для отображения на нашем сайте.
